# EcoCraft3 🌿

A company profile landing page for **Eco Craft** — a creative movement dedicated to making art from eco-friendly materials and supporting earth's sustainability.

🌐 **Live:** [jereminathanael.github.io/EcoCraft-PWD-Kelompok3](https://jereminathanael.github.io/EcoCraft-PWD-Kelompok3/)

---

## About

> *"Eco Craft adalah gerakan kreatif untuk membuat karya seni dari bahan ramah lingkungan dan mendukung keberlanjutan bumi."*

Eco Craft adalah sebuah gerakan kreatif yang bertujuan menciptakan karya seni menggunakan bahan-bahan ramah lingkungan, sekaligus mendukung keberlanjutan bumi. Website ini dibuat sebagai company profile dalam bentuk single-page landing page.

---

## Pages / Sections

| Section | Description |
|---|---|
| Home | Hero section with tagline and CTA |
| About | About Eco Craft movement |
| Products | Showcase of eco-friendly products |
| Group | Group members (Kelompok 3) |
| Contact | Contact information |

## Getting Started

No installation or build tools needed. Just open the file directly in a browser.

```bash
# Clone the repository
git clone https://github.com/jereminathanael/EcoCraft-PWD-Kelompok3.git
cd EcoCraft-PWD-Kelompok3

# Open in browser
open index.html
```

Or simply visit the live site at the link above.
